# ArchitectPro

Professional interactive house construction planner with 2D editor, 3D model, materials calculator, and cost estimator.

## Features
- **2D/3D Split View**: Design in 2D and see the 3D model update live.
- **Switchable Units**: Toggle between Metric (m/cm) and Imperial (ft/in).
- **Material Estimator**: Editable material prices with real-time budget tracking.
- **Multilingual**: Supports English and French.
- **Multi-Floor Support**: Design houses up to 5 floors.

## Technologies
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Three.js (3D Rendering)

## How to Run
Simply open `index.html` in any modern web browser.
